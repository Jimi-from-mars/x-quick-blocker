X Quick Block

🎯
使用方法：
按住 Ctrl 键（Mac用户按 Command），点击任意推文
🎯
How to Use：
Hold Ctrl key (or Command on Mac), click any tweet
✓ 真实屏蔽：插件会模拟手动操作，将用户添加到X官方屏蔽列表中，所有设备同步生效！
✓ Real Blocking：The extension simulates manual operations, adding users to X's official block list, effective across all devices!
⚠️ 注意：
  • 屏蔽操作不可批量撤销
  • 如果屏蔽失败，请尝试手动操作
  • 操作过快可能被X限制
⚠️ Warning：
  • Blocking operations cannot be undone in bulk
  • If blocking fails, try manual operation
  • Too frequent operations may be restricted by X
管理已屏蔽用户请访问：
x.com/settings/blocked/all
Manage blocked users at:
x.com/settings/blocked/all
